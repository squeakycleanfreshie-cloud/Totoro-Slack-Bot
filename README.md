# Totoro Slack Bot

A Totoro-themed Slack bot built with Node.js and Slack Bolt, hosted on Hack Club Nest.

## Commands

| Command | Description |
|---|---|
| `/tot-ping` | Check bot latency |
| `/tot-catfact` | Get a random cat fact |
| `/tot-joke` | Get a random joke |
| `/tot-weather [city]` | Get the weather described in Totoro style |
| `/dsb-help` | Show all available commands |

## Setup

### Prerequisites
- Node.js v24+
- A Slack app with Socket Mode enabled
